## Authority boundary

This skill is procedural only.
It must not introduce architecture or modify invariants.

## Procedure

1) Apply rules from \`docs/DOC_STYLE.md\`.
2) Escape inline backticks as \\\`.
3) Avoid nested code fences. If triple backticks must be referenced, write \`\`\`.
4) Update links and indexes if needed.
