# Design: TailwindCSS CLI Skill Integration

## 1. Skill Folder Structure
The skill will be housed in `Jules/skills/tailwindcss-cli/`.
Subdirectories:
- `assets/`
- `scripts/`
- `references/`

## 2. Skill Metadata
- Name: `tailwindcss-cli`
- Task Group: `operations`

## 3. SKILLS_INDEX.md Integration
Add `tailwindcss-cli/SKILL.md` under the "Operational Skills" section.
Reads first: `PROJECT_PROFILE.md` (Standard for ops).
