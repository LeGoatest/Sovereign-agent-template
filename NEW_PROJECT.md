# New Project Prompt

Use this when opening this repository in an agent environment.

Prompt:

Read this repository and treat it as the canonical project template.

Goal:
Create a new project using the rules, templates, skills, and spec workflow defined in this repository.

Instructions:
- Do not assume any technology, framework, or architecture unless required by the templates.
- Initialize governance, task groups, and skills exactly as defined here.
- Create an initial specification using the provided spec templates.
- Populate requirements, design, and tasks only when user intent is provided.
- Do not execute tasks unless explicitly instructed.

If conflicts exist, refuse.
Follow all rules in this repository.
Begin.
